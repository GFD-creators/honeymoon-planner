# 週グリッドの表示幅拡張 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 週表示のときだけ表示領域 `#view` の最大幅を広げ（min(96vw,1100px)）、PC/タブレットで7列をスクロールなしで見られるようにする。

**Architecture:** CSS で `.view.wide` を定義し、`timeline.js` の `setView` が週表示時に `#view` へ `wide` を付与/除去、`app.js` の `render()` が各描画前に `wide` をリセットする。ロジック変更・テスト追加なし。

**Tech Stack:** Vanilla JS (ES Modules), CSS, GitHub Pages。

---

- [ ] **Step 2: app.js の render() で wide をリセット**

`js/app.js` の `render` 関数を変更。
変更前:

```js
function render() {
  const state = getState();
  viewEl.innerHTML = '';
  renderers[current](viewEl, state);
}
```

変更後:

```js
function render() {
  const state = getState();
  viewEl.innerHTML = '';
  viewEl.classList.remove('wide'); // 既定は narrow（週表示時に timeline が再付与）
  renderers[current](viewEl, state);
}
```

- [ ] **Step 3: timeline.js の setView で wide を付与/除去**

`js/timeline.js` の `setView` 関数を変更。
変更前:

```js
  function setView(v) {
    saveView(v);
    toggle.querySelectorAll('.toggle-btn').forEach((b) =>
      b.classList.toggle('is-active', b.dataset.view === v));
    container.innerHTML = '';
    if (v === 'week') renderWeekView(container, state);
    else renderList(container, state);
  }
```

変更後:

```js
  function setView(v) {
    saveView(v);
    toggle.querySelectorAll('.toggle-btn').forEach((b) =>
      b.classList.toggle('is-active', b.dataset.view === v));
    root.classList.toggle('wide', v === 'week'); // 週表示のみ #view を広げる
    container.innerHTML = '';
    if (v === 'week') renderWeekView(container, state);
    else renderList(container, state);
  }
```

## Task 2: ブラウザ実機確認（Playwright）

**Files:** なし（検証のみ）

- [ ] **Step 1: ローカルサーバ起動**

プロジェクト直下に `tmp-server.cjs` を作成して起動:

```js
const http=require('http'),fs=require('fs'),path=require('path'),root=__dirname;
const t={'.html':'text/html; charset=utf-8','.css':'text/css; charset=utf-8','.js':'text/javascript; charset=utf-8','.json':'application/json; charset=utf-8'};
http.createServer((q,s)=>{let u=decodeURIComponent(q.url.split('?')[0]);if(u==='/')u='/index.html';const fp=path.join(root,u);if(!fp.startsWith(root)){s.writeHead(403);s.end();return;}fs.readFile(fp,(e,d)=>{if(e){s.writeHead(404);s.end();return;}s.writeHead(200,{'Content-Type':t[path.extname(fp)]||'application/octet-stream'});s.end(d);});}).listen(5180,()=>console.log('up'));
```

`curl -s -o /dev/null -w "%{http_code}" http://localhost:5180/index.html` が `200`。

- [ ] **Step 2: 広い画面で週表示の幅を確認**

ビューポートを 1280x900 にして `http://localhost:5180/?trip=test-wide-001` を開く。週表示に切替え、`#view` の実幅が 560px より大きく（約1040〜1100px）なり、7列（`.col-body` が7個）がほぼスクロールなしで収まることを `browser_evaluate` で確認:

```js
() => {
  document.querySelector('.toggle-btn[data-view="week"]').click();
  const view = document.getElementById('view');
  const sc = document.querySelector('.week-scroller');
  return {
    isWide: view.classList.contains('wide'),
    viewWidth: Math.round(view.getBoundingClientRect().width),
    cols: document.querySelectorAll('.col-body').length,
    needsScroll: sc.scrollWidth > sc.clientWidth + 2,
  };
};
```
Expected: isWide=true、viewWidth が 560 より大、cols=7、needsScroll=false（または僅少）
